IG4Tool
Tool to automate and simplify infogroeps administrative tasks

setup
------
	1. do the git clone
	2. install the gems (run bundle)
	3. migrate the database (rake db:migrate)
	4. run the unit tests (rake rspec)
	5. you're set!
